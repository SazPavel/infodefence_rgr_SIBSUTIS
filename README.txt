Протокол доказательства с нулевым значением для задачи "Раскраска графа"
tmp/graph - заданный граф
в первой строке содержатся количество вершин графа и количество ребер
в последующих строках информация о ребрах графа
(номера вершин, соединяемых ребром)
tmp/graph2 - результат работы ./color  - перечисление цвета вершин графа
./color - раскрашивает граф
./client - запрашивает у сервера список цвета вершин
./server - шифрует цвета вершин графа с помощью шифра вернама и отправляет их на проверку ./client
./client выбирает случайное ребро, шифрует номера вершин, соединяемых этим графом с помощью шифра вернама и отправляет их ./server
./server расшифровывает полученные вершины и возвращает их ./client
./client расшифровывает вершины своим шифром и проверяет что полученные цвета различны
в случает ошибки сообщает об этом
количество проверок неограниченно, при каждой проверке цвета вершин шифруются новым ключом
